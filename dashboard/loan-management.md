# 8.1 Loan Management

Borrowers can:
- View all loan requests made
- Track status: Pending, Approved, Active, Repaid, Defaulted
- Monitor:
  - Loan amount
  - Interest rate
  - Repayment deadline
  - Collateral locked in XLM
- Repay loans directly from the dashboard
- See TRBT score evolution after each repayment
